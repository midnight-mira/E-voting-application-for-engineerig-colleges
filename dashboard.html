About

Our “Digi Election” aims to make the online student voting process easy and convenient for engineering colleges. This system provides online voter registration forms for students where students get registered and are allowed to log in as voters. Each registered user has a password to log in. The system provides an interactive platform where voters and candidates can interact. The purpose of this Digi-Election is to present an online voting system ,to be applied to engineering colleges for CRs elections. Several security measures are integrated into the system in order to achieve an enhanced, speedy and accurate performance.
